# HanNeungGum_Place
서울역과 동서울터미널에서 한능검 시험장까지의 대중교통을 이용한 시간을 측정 후 html 파일로 생성합니다.

네이버 Maps의 geocoding API와 ODsay의 API를 이용합니다.

py 파일에서 NAVER_CLIENT_ID, NAVER_CLIENT_SECRET, ODSAY_API_KEY를 본인의 KEY 값으로 수정하세요.

Excel 파일의 이름은 "한능검_시험장"이며, 여부가 "가능"인 시험장만 선별하여 시험장까지의 소요 시간을 계산합니다.

완성된 html 파일의 이름은 "시험장_소요시간"입니다.
